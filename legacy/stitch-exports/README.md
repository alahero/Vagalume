# Archivo: exports de Google Stitch

Esta carpeta conserva **solo como referencia** los HTML, capturas, ZIPs y scripts que salieron del flujo Stitch. **No forma parte del build** de la app en `web/`.

- El sitio que se despliega vive en `web/` (`npm run dev` / `npm run build`).
- Los medios que carga el navegador en producción deben estar en `web/public/` (por ejemplo `public/media/`, `public/events/`).

Si generas un export nuevo, colócalo aquí o actualiza `stitch-export/` para no mezclarlo con el código React.
